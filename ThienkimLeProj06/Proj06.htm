PROGRAM SPECIFICATIONS

The zip file that you downloaded contains a file named Proj06Runner.txt. Rename that file to Proj06Runner.java.

Place that file in the same folder as the file named Proj06.java that was also in the zip file that you downloaded.

Compile the file named Proj06Runner.java.

Attempt to compile the file named Proj06.java. You will find that it won't compile.

Make corrections as necessary to the file named Proj06Runner.java to cause both files to compile.  Note that you must not modify code in the file named Proj06.java.

Run the file named Proj06.java. It must produce the following output on the command-line screen. 

I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Display your name here

Be sure to display your actual name where indicated.

Encapsulate your file named Proj06Runner.java in a zip file and submit it via Blackboard.
